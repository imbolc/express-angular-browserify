express-angular-browserify
==========================

Seed app with next stack:

- angular.js in the browser
- node.js server
- express.js web framework
- ejs / jade templating
- browserify modules
- url-reversing

Install
-------
    sudo npm install nodemon browserify -g
    npm install

Start dev server
----------------
    nodemon app.js

Start prod server
-----------------
    npm start
